
1. notifications

notification terjadi ketika
1. agent mengerjakan
   notif ke : admin dan customer
2. chat ke customer ke agent / sebaliknya
   notif ke : customer dan agent yang menerima chat
3. chat ke agent ke agent
   notif ke : agent yang menerima chat
4. agent menyelesaikan pekerjaan
   notif ke : admin dan customer
5. customer memberikan rating
   notif ke : agent
